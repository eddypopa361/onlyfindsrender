Corectează tema de culori. Cerință: toate accentele brand (inclusiv cele rămase mov, violet, purple, indigo, magenta, “brand”, “accent”, #ff6347, #ff6347cc și orice alt hex mov) trebuie înlocuite cu noua culoare primară #cb2410 (și doar variațiile sale de lumină/umbră).

Specificații stricte:
- Primary solid: #cb2410
- Text pe primary și texte accent: #fefefe (alb).
- Background principal / suprafețe dark: #000000, cu nivele secundare #111111, #1a1a1a dacă sunt necesare.
- Orice buton, link accent, icon highlight, badge, toggle activ, focus ring colorat → bază #cb2410 (fără #ff6347, fără mov).
- Orice gradient care folosea mov/branding vechi sau #ff6347* devine:
  linear-gradient(135deg, #cb2410 0%, #b11f0d 55%, #ff9988 100%)
  + fallback: background-color: #cb2410;
- Glow / shadow accent: folosește rgba(203,36,16,0.45) sau pentru inele focus două straturi:
  0 0 0 2px #000000, 0 0 0 4px rgba(203,36,16,0.55).
- Dacă există tokeni / var CSS (ex: --color-accent, --brand, --primary-500), reasignează-i la noua paletă; elimină tokenii mov nefolosiți.

   - `colors.primary` = scala calibrată (o poți regenera, păstrând #cb2410 drept 500).
